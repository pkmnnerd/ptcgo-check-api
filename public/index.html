Pokemon TCG code checker

This is a simple service that checks Pokemon TCG codes. It proxies to the pokemon.com service to get the values, but uses an easier to use API and without any CORS restrictions so other websites can retrieve the information.

This service requires a Pokemon.com account to look up the code. You'll need to retrieve your account cookies using the steps below.

  1. In a desktop Chrome browser, go to https://pokemon.com/redeem and log in.
  2. Complete the Captcha.
  3. Paste in a random string ("test" for example) and verify the code.
  4. Click on the lock on the left of the address bar and click "Cookies".
  5. Expand www.pokemon.com > Cookies and click on op_session_id.
  6. Copy the value of "Content" and paste it below.
